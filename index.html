Documentation - Application de Gestion des Équipements

Bienvenue dans l'application de gestion des équipements. Cette application permet de gérer, suivre et organiser vos équipements facilement, avec un accès rapide aux informations et statistiques.

Gestion des équipements

Ajouter / Modifier / Supprimer

  • Ajouter : Remplir le formulaire avec toutes les informations obligatoires pour créer un nouvel équipement.
  • Modifier : Sélectionner un équipement existant et changer les champs nécessaires.
  • Supprimer : Supprimer un équipement en cliquant sur l'option correspondante. Une confirmation sera demandée.

Champs obligatoires et erreurs

Les champs obligatoires pour chaque équipement sont :

  • Nom
  • Série
  • Type
  • Date d'achat (format JJ/MM/AAAA)
  • Marque
  • Localisation
  • Modèle
  • Statut

⚠️ Si un champ obligatoire est vide ou contient un texte placeholder comme "entrer" ou "définir", une erreur sera affichée.

FAQ / erreurs fréquentes

  • Erreur : "Le champ Nom est obligatoire." → Assurez-vous de remplir tous les champs obligatoires.
  • Erreur : "Le numero de serie existe deja." → Assurez-vous de declarer des numeros de series uniques.
  • Erreur : "La date d'achat doit être au format JJ/MM/AAAA." → Vérifiez le format de la date.
  • Erreur : "Aucune modification détectée." → Vérifiez que vous avez modifié au moins un champ avant de sauvegarder.
  • Erreur : "Équipement introuvable." → Vérifiez l'ID de l'équipement que vous essayez de consulter.
